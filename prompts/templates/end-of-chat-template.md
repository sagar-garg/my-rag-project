# End Of Chat Template

Before we end this chat:

1. Update durable learnings in the right docs under `docs/`.
2. Update `prompts/current/project-state.md` to reflect the current repo state.
3. Update `prompts/current/next-chat.md` with the best next prompt.
4. Update `prompts/current/backlog.md` if priorities changed.
5. If something major was completed, add a short checkpoint under `docs/checkpoints/`.
6. Summarize:
   - what changed
   - what we learned
   - the next natural step

Do not rewrite archived prompts. Create new archive entries only when a prompt set is being retired.
