## Observations

- **The baseline is saturated.** 4/4 with every first hit at rank 1 means this
  eval set cannot show improvement — any retrieval feature added now would
  measure 100% → 100%. Expanding the eval set (roadmap item 2) is a
  prerequisite for all feature work, not a nice-to-have.
- **The questions are too easy for a dense retriever.** Two of the four contain
  the literal words "Chapter 6", and all four reuse the chapters' own
  vocabulary ("evaluation-driven development", "prompting", "finetuning").
  Harder questions should avoid chapter names and paraphrase concepts instead
  of quoting them.
- **Question 3 is the only crack in the wall:** purity 2/4, with two
  Chapter 5 chunks retrieved for a "context construction" question. The
  prompt-engineering chapter shares that vocabulary heavily — cross-chapter
  confusion between Ch5 and Ch6 is the most promising axis for harder eval
  questions.
